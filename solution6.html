Ordered list of HTML tags

  1. <p>:The paragraph tag defines a paragraph.
  2. <hr>:The hr tag defines a thematic break in an HTML page
  3. <a>:The a tag defines a hyperlink, which is used to link from one page to another.
  4. <b>:The b tag specifies bold text without any extra importance.
  5. <ul>:The ul tag defines an unordered (bulleted) list.
